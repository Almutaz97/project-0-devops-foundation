# Project 0 — DevOps Foundation Setup

## Goal

Prepare a clean DevOps working environment and practice the basic workflow discipline required for future DevOps projects.

This project is the foundation before moving into Linux servers, web servers, Dockerized applications, Kubernetes, CI/CD, and cloud infrastructure.

## Environment

- OS: Ubuntu VM
- User: ubuntu
- Git: installed and configured
- SSH: configured with GitHub authentication
- Docker: installed and active
- Docker Compose: installed
- Repository remote: GitHub over SSH
